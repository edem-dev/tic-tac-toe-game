# Ticky

A modern, responsive Ticky game built with React, TypeScript, and Tailwind CSS. Featuring single-player against AI, real-time online multiplayer, and PWA support.

## Features

- **Single Player**: Challenge an AI with three difficulty levels: Easy, Medium, and Expert.
- **Multiplayer**: Play with friends online using room codes, powered by PeerJS.
- **Score Tracking**: Keep track of wins and draws during your gaming session.
- **Modern UI**: Clean, dark-themed interface with smooth animations.
- **PWA Support**: Install the app on your device for an app-like experience and offline play.

## Technologies Used

- **React**: UI library
- **TypeScript**: Type safety
- **Tailwind CSS**: Styling
- **PeerJS**: Peer-to-peer connectivity for multiplayer
- **Lucide React**: Icon library
- **Vite**: Build tool

## Getting Started

### Prerequisites

- Node.js (v16 or higher)
- npm or yarn

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/edem-dev/tic-tac-toe-game.git
   cd tic-tac-toe-game
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Run the development server:
   ```bash
   npm run dev
   ```

4. Open [http://localhost:5173](http://localhost:5173) in your browser.
